# 幾何物件與解析幾何運算說明

本專案以解析幾何為基礎，使用物件導向方式實作「點、線、圓、三角形」，
並進行交點計算、垂線構造、畢氏定理驗證，以及幾何變換。

---

## 1. 點（Point）

### 數學定義
平面上一點以座標表示：
\[
P = (x, y)
\]

### 幾何運算
- 距離：
\[
d = \sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}
\]
- 平移：
\[
(x', y') = (x + dx, y + dy)
\]
- 縮放（以中心 C 為基準）：
\[
x' = x_c + k(x - x_c)
\]
\[
y' = y_c + k(y - y_c)
\]
- 旋轉（角度 θ）：
\[
\begin{pmatrix}
x' \\ y'
\end{pmatrix}
=
\begin{pmatrix}
\cos\theta & -\sin\theta \\
\sin\theta & \cos\theta
\end{pmatrix}
\begin{pmatrix}
x-x_c \\ y-y_c
\end{pmatrix}
+
\begin{pmatrix}
x_c \\ y_c
\end{pmatrix}
\]

---

## 2. 直線（Line）

### 數學表示
使用一般式：
\[
ax + by + c = 0
\]

### 兩點決定一直線
\[
a = y_2 - y_1,\quad b = x_1 - x_2
\]

### 兩直線交點
解聯立方程：
\[
\begin{cases}
a_1x + b_1y + c_1 = 0 \\
a_2x + b_2y + c_2 = 0
\end{cases}
\]

---

## 3. 圓（Circle）

### 數學表示
\[
(x - h)^2 + (y - k)^2 = r^2
\]

### 兩圓交點
- 利用圓心距判斷交點數量
- 使用三角形與餘弦定理求交點座標

### 圓與直線交點
- 計算圓心到直線距離：
\[
d = \frac{|ax_0 + by_0 + c|}{\sqrt{a^2+b^2}}
\]
- 若 \(d \le r\)，則有交點

---

## 4. 垂線與垂足

### 垂線性質
若直線為：
\[
ax + by + c = 0
\]
則垂直線為：
\[
bx - ay + c' = 0
\]

### 垂足
垂足為原直線與垂線的交點。

---

## 5. 畢氏定理驗證

透過：
- 直線上的一點 A
- 線外一點 P
- 垂足 H

形成直角三角形，驗證：
\[
PH^2 + HA^2 = PA^2
\]

---

## 6. 三角形（Triangle）

### 定義
由三個點構成：
\[
\triangle ABC
\]

### 幾何變換
- 平移
- 縮放
- 旋轉  
均由其頂點的點變換組成。

---

## 7. 幾何變換的統一性

所有幾何物件的變換，本質上都是對「點」做變換，
再由點重新構成線、圓、三角形。
