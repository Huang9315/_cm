定理說明
這兩個定理是現代通訊系統的基石，主要解決「如何在雜訊干擾下進行可靠通訊」以及「通訊速度的物理極限」問題。
1. 夏農信道編碼定理 (Shannon's Channel Coding Theorem)
這個定理又稱為「雜訊信道編碼定理」（Noisy-channel coding theorem）。它給出了一個非常樂觀且反直覺的結論：
核心概念： 即使信道（傳輸媒介，如電線、無線電波）有雜訊干擾，只要你的資訊傳輸速率 (R) 小於該信道的容量 (Capacity, C)，就存在一種編碼方式，可以讓錯誤率（Error Rate）隨著編碼長度的增加而趨近於 0。
白話文： 不管雜訊多大（只要不是無限大），只要我們講話的速度控制在某個極限內，並且用足夠聰明的編碼（例如多講幾次或用漢明碼這類更高級的糾錯碼），對方一定能幾乎完美地聽懂。
數學表達：
若 $R < C$，存在編碼使誤差 $P_e \to 0$。
若 $R > C$，可靠通訊是不可能的，誤差無法避免。
2. 夏農-哈特利定理 (Shannon–Hartley Theorem)
這個定理具體化了上面的「信道容量 C」是多少。它結合了物理限制（頻寬）與信號強度。
公式：

$$C = B \log_2 \left( 1 + \frac{S}{N} \right)$$
$C$：信道容量 (Channel Capacity)，單位是 bits per second (bps)。
$B$：頻寬 (Bandwidth)，單位是赫茲 (Hz)。
$S$：平均信號功率 (Signal power)。
$N$：平均雜訊功率 (Noise power)。
$S/N$：信噪比 (Signal-to-Noise Ratio)。
核心意義：
頻寬換速度： 增加頻寬 $B$ 可以線性地增加傳輸速度（路變寬了，車流量變大）。
功率換速度： 增加信號強度 $S$ 也可以增加速度，但效果是對數增長（Logarithmic）。這意味著當信噪比已經很高時，再拼命增加發射功率，對速度的提升很有限（邊際效應遞減）。
無中生有是不可能的： 這個定理定義了 5G、Wi-Fi 或任何通訊系統的物理極限。你無法超越這個公式算出的最大速度。
