使用數值方法求解特徵根 (numpy.roots)：
常係數齊次 ODE $a_n y^{(n)} + \dots + a_0 y = 0$ 的解法核心在於求解其特徵多項式 $a_n \lambda^n + \dots + a_0 = 0$。
Python 的 numpy.roots 函數是一個強大且方便的工具，可以找出多項式的所有根（包含實數與複數）。
處理浮點數誤差與分組 (Tolerance)：
電腦運算中，1.0 和 0.999999999 可能被視為不同，但在數學求解中它們應視為同一個根（重根）。
此外，純實數的解在計算過程中可能會產生極小的虛部（如 2 + 1e-15j）。
因此，程式設定了一個 tol (tolerance, 容許值)，用來判定：
兩個根是否足夠接近以視為「重根」。
虛部是否足夠小以視為「實根」。
實部是否足夠小以視為 0 (例如在 $e^{0x}$ 的簡化顯示時)。
根的分類與通解構建：
程式將根分為兩大類處理：
實根 $r$ (重數 $m$)：對應的解為 $e^{rx}, xe^{rx}, \dots, x^{m-1}e^{rx}$。
複數根 $\alpha \pm \beta i$ (重數 $m$)：對應的解為 $e^{\alpha x}\cos(\beta x), e^{\alpha x}\sin(\beta x)$，若有重根則再乘上 $x, x^2 \dots$。
對於複數根，我們只需要抓取共軛對中的其中一個（通常取虛部為正者），即可生成對應的 $\sin$ 與 $\cos$ 兩項。
